# Smart Contract

Hardhat project for smart contract development.

## Setup

```bash
npm install
```

## Compile

```bash
npm run compile
```

## Test

```bash
npm test
```

## Deploy

```bash
npm run deploy
```

## Start Local Node

```bash
npm run node
```

## Structure

- `contracts/` - Solidity smart contracts
- `test/` - Test files
- `scripts/` - Deployment scripts
- `artifacts/` - Compiled contracts (generated)
